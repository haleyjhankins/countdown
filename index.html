Responsive Layout using 2 Media Queries

Please resize the browser window's width to see how the image above rescales to fit the screen resolution (browser window width). The trick in making images responsive is to set the width to 100% (not pixels). This will keep the image as large as the container it has been placed within. In this case the container is a blue section tag set to 70% of the body. The white body area has been set to 80% of the browser window. You will also notice this paragraph responds to the available width by reflowing from maximum 3 columns (over 1000px wide - large desktop monitors)down to media querie set to trip down to 2 columns for tablets (under 1000px) and another down to 1 column for smart phones (below 800px).
